# Vectrax Landing Page

A premium, minimal landing page for the Vectrax Android app. Built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Scroll-linked zoom effect** – Hero mockup zooms out smoothly as you scroll
- **Google/Stripe-style minimalism** – Pure white background, generous spacing
- **Supabase Database Status** – Neon-themed status indicator with pulse animation
- **Stripe-style buttons** – Subtle shadows, 8px rounded corners, clean hover states

## Tech Stack

- Next.js 14
- Tailwind CSS
- Framer Motion
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
